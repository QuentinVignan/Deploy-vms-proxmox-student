# Install Deps

```
python3 -m pip install -r ./requirements.txt
```

# Execution

for generate config file run :

```
python3 main.py -it
```

for create vm run :
```
python3 main.py --create --config ./config/filename.yaml
```

for delete vm run :
```
python3 main.py --delete --config ./config/filename.yaml
```
